

PREDICTOR RATING 

Rating predictor is a python model made in python that rates movies based on what people 
commented about on twitter. It works based on a collection of data obtained from twitter 
developers. The method cleans the data and detects de level of polarity and subjectivity.

## Getting Started

To start working with this program you just need to download the python notebook and 
run your own set of data.

### Prerequisites

pandas and numpy packages

##To download a dataset from twitter you can download the getTweets file 
Download the movie or show you are interested of knowing the ratings.
